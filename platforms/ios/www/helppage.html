This is the eCook help page. Here you will find instructions on how to use the app and its various functions.

Recipe List Page: This page displays a list of saved recipes. On the top right of the page is a button labeled "Add". Tapping this will open up a menu that asks you to select a format for a new recipe. Possible choices are "Input Text", "Take a Photo", and "Import from Website". If you decide not to add a recipe after all, tap "Cancel".
