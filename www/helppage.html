This is the eCook help page. Here you will find instructions on how to use the app and its various functions.

Adding Recipes:
On the top right of the recipe list page is a button labeled "Add". Tapping this will open up a menu that asks you to select a format for a new recipe. Possible choices are "Input Text", "Take a Photo", and "Import from Website". If you decide not to add a recipe after all, tap "Cancel".

Editing Recipes:
To edit a plain text recipe, tap "Edit" on top-left of the recipe list page. This should show you the details of that recipe. From this screen, you should be able to edit any part of the recipe by tapping the relevant text box and using the virtual keyboard. Make sure to press "Save" to keep any changes you make.

Viewing Recipes:
To view a recipe, simply select one on the recipe list page. This should display the details of the selected recipe.
